# Wide Wolrd Importers 

## Objetivo del proyecto

Diseñar una solución integral de análisis empresarial utilizando SQL Server y Power BI, basada en la base de datos Wide World Importers, con enfoque en los procesos de ventas y compras. El objetivo principal es transformar datos transaccionales en información estratégica mediante un modelo dimensional optimizado, permitiendo evaluar el desempeño comercial, la rentabilidad, la evolución del gasto y la eficiencia en la gestión de proveedores.

Los dashboards interactivos desarrollados permiten analizar indicadores clave como ingresos, margen, ticket promedio, concentración de clientes, variación mensual del gasto, desempeño de proveedores y relación entre compras y ventas. Esta solución facilita la identificación de patrones de crecimiento, riesgos de dependencia, impacto de costos en la rentabilidad y oportunidades de optimización operativa.

Como estudiante de Ingeniería en Sistemas con orientación hacia el análisis y la ciencia de datos, desarrollé este proyecto con el propósito de fortalecer mis habilidades en modelado estrella, consultas avanzadas en SQL, creación de medidas en DAX, diseño de KPIs ejecutivos y construcción de visualizaciones enfocadas en la toma de decisiones empresariales.

Este proyecto representa una aplicación práctica de business intelligence, donde la integración entre base de datos y herramientas analíticas permite convertir información operativa en conocimiento accionable, consolidando una base sólida para futuros desarrollos en analítica avanzada y data-driven decision making.

# KPIs que este proyecto puede responder 

### Rendimiento general

- ¿Cuánto vendimos en total?

- ¿Cuánto ganamos?

- ¿Cuál es el margen global?

- ¿Cuántas órdenes se generaron?

- ¿Cuál es el ticket promedio?

### Crecimiento

- ¿Las ventas están creciendo?

- ¿El crecimiento es constante?

- ¿Las ganancias crecen al mismo ritmo que las ventas?

- ¿El número de órdenes aumenta o disminuye?

### Producto

- ¿Qué categorías venden más?

- ¿Qué categorías son más rentables?

- ¿Qué productos generan más ganancias?

- ¿Existe relación entre volumen vendido y rentabilidad?

### Clientes

- ¿Qué tipo de cliente genera más revenue?

- ¿Hay concentración en un solo segmento?

- ¿Quiénes son los Top 10 clientes?

### Operación

- ¿Qué porcentaje de órdenes se finalizan?
